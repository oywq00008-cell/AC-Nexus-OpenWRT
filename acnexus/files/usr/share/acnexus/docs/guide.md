## 它能做什么？

| 功能 | 一句话解释 |
|------|-----------|
| 🎮 **手机/电脑遥控空调** | 不用找遥控器，打开浏览器就能开关空调、调温度、切换模式 |
| 🌤️ **看天气自动调温** | 外面 35°C → 自动开 26°C 制冷；外面 10°C → 自动开 28°C 制热 |
| 🌀 **台风来了自动关** | 台风靠近时按风速三级判定，自动关空调保平安 |
| ⏰ **多段定时开关** | 每天可设多组开关机时间，支持按工作日/周末分组 |
| 📋 **多模板切换** | 夏季模板一套时间、冬季另一套时间，一键切换 |
| 📡 **博联 + 米家双协议** | 支持博联 RM 系列红外遥控和米家空调伴侣（MIoT） |
| ⚡ **毫秒级响应** | Lua 接管高频 API，告别 Python 冷启动的等待 |

> **打个比方**：普通空调是手动挡汽车，装上这个插件就变成了自动挡——你只需要告诉它"我想要 26°C"，它会根据外面天气自己决定开制冷还是制热。

## 准备工作

你需要准备：

| 物品 | 说明 | 大概费用 |
|------|------|-----------|
| 🖥️ **OpenWRT 路由器** | 已刷 OpenWRT 系统的路由器 | 已有 |
| 📡 **博联 RM 或 米家空调伴侣** | 博联发射红外信号控制空调；米家通过 Wi-Fi 直连 | ~100 元 |
| 🌤️ **天气 API Key** | 百度或和风天气，免费的天气数据接口（下面有申请教程） | 免费 |

> **原理**：手机/电脑 → 路由器（插件） → 博联 RM（红外）/ 米家空调伴侣（Wi-Fi） → 空调。

## 安装插件

### 方法一：LuCI 网页上传 IPK（推荐）

从 [GitHub Releases](https://github.com/oywq00008-cell/AC-Nexus-OpenWRT/releases) 下载 `acnexus_openwrt_v5.0.0_all.ipk`：

1. 浏览器打开路由器后台
2. **系统 → 软件包 → 上传软件包**
3. 选择下载的 `.ipk` 文件
4. 等待安装完成，刷新页面即可

### 方法二：SCP 手动安装（兜底）

如果 IPK 上传失败，下载包含 `install.sh` 的完整包：

```bash
# 上传到路由器
scp install.sh acnexus_*.ipk root@你的路由器IP:/tmp/

# SSH 登录执行
ssh root@你的路由器IP
cd /tmp && bash install.sh
```

### 打开控制面板

浏览器访问 `http://路由器IP/cgi-bin/luci/admin/services/acnexus`，或在 LuCI 菜单点击 **服务 → AC-Nexus**。

## 申请天气 API（免费）

插件需要"外面多少度"才能自动调温。支持**百度天气**和**和风天气**双源，可以只申请一个，建议两个都申请（一个挂了自动切换）。

### 和风天气（推荐，注册最简单）

**1. 注册账号** → [和风天气控制台](https://console.qweather.com)

**2. 创建项目** → 左侧「项目管理」→「创建项目」，选**免费订阅**。

**3. 获取 Key** → 创建成功后在项目列表看到 API Key。

**4. 填入插件** → 在设置页填入 Key 和 Host 地址。

### 百度天气（备选）

**1. 注册** → [百度地图开放平台](https://lbsyun.baidu.com/apiconsole/key)

**2. 创建应用** → 选「服务端」，获取 AK。

**3. 填入插件** → 设置页的「百度天气 Key」。

> **提示**：两个 Key 都填好之后，插件默认用百度，百度失败自动回退和风。只填一个也能用。

## 连接空调

### 博联 RM 系列

**第一步：扫描设备**

把博联 RM 插上电，确保和路由器在同一网络。在控制面板点击 **🔍 扫描**，插件会自动发现设备。

**第二步：配对空调**

扫描成功后点击 **⚙ 设备设置**：选择空调品牌（14 种红外协议，覆盖 16 个品牌），保存。

**第三步：发码测试**

回到控制面板，选好模式和温度，点发送按钮，空调应有反应。

> **没反应？** 检查博联 RM 是否对着空调、中间无遮挡。红外需要直线对准。

### 米家空调伴侣

**第一步：生成二维码**

点击工具栏 **＋** 按钮，在弹出的米家设备弹窗中点击「生成二维码」。

**第二步：扫码授权**

用米家 App 扫描二维码，授权插件访问你的米家设备列表。

**第三步：选择设备**

授权后弹窗会列出你的空调伴侣，勾选并确认添加即可。

> 米家设备通过 Wi-Fi 直连，不需要对准空调，也不受红外遮挡影响。


### 操作流程

1. 先选好模式、温度、风速
2. 点中间发送按钮
3. 空调响应，底部显示"✅ 已发送"

> **就像点外卖**：先选菜（模式、温度、风速），再点下单（发送按钮）。

### 环境信息

控制面板下方显示当前环境数据：

- **室外温度、天气、湿度**：来自天气 API
- **风暴距离**：有台风时显示距离 + 等级 emoji（🟢🟠🔴）
- **位置**：当前设置的城市名

## 设备改名

如果你有多台设备，可以通过改名区分：

1. 顶部下拉框选择要改名的设备
2. 点击 **⚙ 设备设置**
3. 在"改名"输入框输入新名字（如"客厅空调"）
4. 保存

> 名字不能为空，也不能和其他设备重名。

## 定时和自动调温（进阶）

### 什么是定时模板？

给空调定一个"作息表"。每天可以有多段时间，按工作日/周末分组：

- **工作日**：早上 8 点开、晚上 6 点关
- **周末**：早上 10 点开、晚上 11 点关

可创建多个模板（如"夏季"、"冬季"），切换设备时一键切换。

### 怎么设置？

1. 点击 ⏰ 打开定时弹窗
2. 开启「启用定时任务」
3. 在下拉框选择或新建模板
4. 勾选生效日期（周一~周日）
5. 填写开机/关机时间（可添加多段，勾选框控制启停）
6. 如需自动调温，开启「自动调温」开关
7. 保存设置

> 定时关闭后自动调温不受影响；自动调温关闭后定时也不受影响。两者独立控制。

## 温度规则

### 它是干什么的？

温度规则是"大脑"——告诉插件外面多热该开制冷、多冷该开制热。**定时开关和自动调温都会用到规则。**

### 在哪里设置？

点击工具栏 **🌡** 打开规则编辑弹窗。

### 规则怎么写？

一条规则包含：室外温度区间 → 目标温度 + 模式：

```
室外 30°C ~ 35°C  → 26°C [制冷]
```

| 部分 | 意思 | 举例 |
|------|------|------|
| 室外温度范围 | 外面在这个区间时触发 | 30°C ~ 35°C |
| 目标温度 | 空调开到多少度 | 26°C |
| 模式 | 制冷/制热/除湿/送风/关机 | 制冷 |

### 默认规则

插件自带 6 条默认规则：

| 室外温度 | 做什么 |
|---------|--------|
| 36°C 以上 | 制冷 24°C |
| 33°C ~ 35°C | 制冷 25°C |
| 30°C ~ 32°C | 制冷 26°C |
| 25°C ~ 29°C | 制冷 27°C |
| 18°C ~ 24°C | 关机（舒适，不需空调） |
| 17°C 以下 | 制热 28°C |

## 常见问题

### Q: 扫描不到博联设备？

A:
1. 检查博联 RM 是否通电
2. 确保路由器和博联在同一网络（同一广播域）
3. 多等几秒再试

### Q: 多台同型号设备分不清？

A: 在设备设置弹窗里用"改名"功能给每台起昵称（见[设备改名](#设备改名)）。

### Q: 空调品牌不在列表里？

A: 目前支持 14 种红外协议、16 个可选品牌：格力、美的、华凌、小米、海尔、海信、日立、大金、三菱、松下、富士通、奥克斯、巴鲁、开利、现代、Fuego。如果品牌不在列表中，程序默认尝试格力协议。

### Q: 天气数据不显示？

A: 需要先申请天气 API Key 并填入设置页。百度/和风都填好之后自动双源切换。

### Q: 和风 API 提示 401 错误？

A:
1. 确认 API Key 没有复制错
2. 确认 Host 地址正确
3. 新注册的 Key 可能有几分钟延迟

### Q: 台风数据不显示？

A: 台风只在西北太平洋有风暴时才有数据（一般 6~10 月台风季），其他时间显示 `--` 是正常的。

### Q: 米家设备发送超时？

A: 米家设备需要路由器能访问外网（MIoT 协议依赖小米云服务）。确保路由器已联网。

### Q: 设备设置里搜索框文字看不清？

A: 是 LuCI 主题样式冲突，不影响使用。输入城市名搜索即可。
