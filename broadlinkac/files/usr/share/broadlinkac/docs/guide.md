## 它能做什么？

| 功能 | 一句话解释 |
|------|-----------|
| 🎮 **手机/电脑遥控空调** | 不用找遥控器，打开浏览器就能开关空调、调温度、切换模式 |
| 🌤️ **看天气自动调温** | 外面 35°C → 自动开 26°C 制冷；外面 10°C → 自动开 28°C 制热 |
| 🌀 **台风来了自动关** | 台风离你不到 100 公里，自动关空调保平安 |
| ⏰ **定时开关** | 像闹钟一样，到点自动开、到点自动关 |
| 📋 **多场景模板** | 工作日一套时间、周末另一套时间，自由切换 |
| 🏷️ **设备改名** | 多台同型号博联设备也能分清，支持自定义昵称 |

> **打个比方**：普通空调是手动挡汽车，装上这个插件就变成了自动挡——你只需要告诉它"我想要 26°C"，它会根据外面天气自己决定开制冷还是制热。

## 准备工作

你需要三样东西：

| 物品 | 说明 | 大概多少钱 |
|------|------|-----------|
| 🖥️ **OpenWRT 路由器** | GL-MT6000 或其他支持 OpenWRT 的路由器 | 已有 |
| 📡 **博联 RM 红外遥控器** | 一个小盒子，能发射红外信号控制空调（相当于万能遥控器） | ~100 元 |
| 🌤️ **和风天气 API Key** | 免费的天气数据接口，用来获取室外温度（下面有申请教程） | 免费 |

> **原理很简单**：手机/电脑 → 路由器（插件） → 博联 RM（红外） → 空调。就像你对着空调按遥控器，只不过现在是用手机发指令。

## 安装插件

### 第一步：下载安装包

从 [GitHub Releases](https://github.com/oywq00008-cell/BroadlinkAC-OpenWRT/releases) 下载：
- `broadlinkac_xxx.ipk`（插件包）
- `install.sh`（安装脚本）

### 第二步：上传到路由器

把两个文件传到路由器的 `/tmp/` 目录（可以用 WinSCP 或 scp 命令）。

### 第三步：安装

SSH 登录路由器，执行：

```bash
cd /tmp
bash install.sh
```

### 第四步：打开控制面板

浏览器访问路由器的 LuCI 界面（通常是 `http://192.168.8.1`），点击 **服务 → Broadlink 空调控制**。

## 申请天气 API（免费）

插件需要知道"外面多少度"才能自动调温。这个数据来自和风天气，**完全免费**，每月 5 万次查询足够家用。

### 申请步骤（5 分钟搞定）

**1. 注册账号**

打开 👉 [和风天气控制台](https://console.qweather.com)，用手机号或邮箱注册。

**2. 创建项目**

登录后点左侧菜单「项目管理」→ 右上角「创建项目」：
- 项目名称随便填，比如"我家空调"
- 选择 **免费订阅**

**3. 获取 API Key**

创建成功后，在项目列表里能看到一串字母数字，这就是你的 **API Key**（相当于密码，不要告诉别人）。

**4. 填入插件**

在插件的 **设置页**（服务 → Broadlink 空调控制 → 设置）里：
- 和风天气 API Key：粘贴刚才复制的 Key
- 和风天气 Host：填写你从和风控制台获取的 API Host 地址（格式类似 `xxx.re.qweatherapi.com`）
- 位置：点击搜索框输入你的城市名，选择后自动填入经纬度

> **提示**：百度天气也可以申请（[百度地图开放平台](https://lbsyun.baidu.com/apiconsole/key)），同样免费。建议两个都申请，一个挂了另一个顶上去。

## 连接空调

### 第一步：扫描设备

把博联 RM 插上电，确保和路由器在同一个网络，然后：

在控制面板顶部工具栏点击 **🔍 扫描设备**，插件会自动发现博联 RM。

> 如果你有多台同型号的博联设备（比如 3 个 RM4 mini），插件会自动给每台分配不同的名字，比如"RM4 mini"、"RM4 mini (2)"、"RM4 mini (3)"，方便区分。

### 第二步：配对空调

扫描成功后，点击顶部 **🔧 设置**：
- 选择你的空调品牌（支持 16 个品牌：格力、美的、华凌、小米、海尔、海信、日立、大金、三菱、松下、富士通、奥克斯、巴鲁、开利、现代、Fuego）
- 点击保存

### 第三步：试试能不能控制

回到控制面板主界面，点中间大圆按钮，空调应该就有反应了。

> **如果没反应**：检查博联 RM 是不是对着空调（中间没有遮挡物），红外信号和电视遥控器一样，需要对准才行。

## 控制面板详解

打开 LuCI → 服务 → Broadlink 空调控制，看到的主界面就是控制面板：

```
┌──────────────────────────────────────┐
│ RM4 mini                ● 在线       │  ← 设备名称和在线状态
│                                      │
│             ▶ (大圆按钮)              │  ← 点一下发送当前设置
│                                      │
│    电源关  制冷  制热  除湿  送风      │  ← 选择开关和模式
│                                      │
│              26°C                    │  ← 滑动选择温度
│     16°  20°  24°  28°  30°         │
│                                      │
│     自动   低    中    高             │  ← 风速
└──────────────────────────────────────┘
```

### 操作流程

1. 先在下面选好"制冷"、温度 26°C、风速"自动"
2. 点中间大圆按钮，指令就发出去了
3. 底部会短暂显示"✅ 已发送: 制冷 26°C 自动"

> **就像点外卖**：先选菜（模式、温度、风速），再点下单（大圆按钮）。

### 环境信息

控制面板下方会显示当前环境数据：

```
环境信息                      📍 华湖镇
  32°C    晴    65%    --km
```

- **室外温度**和**天气状态**（晴/阴/雨等）来自和风天气
- **湿度**同样来自天气 API
- **风暴距离**：有台风时显示距离，无台风时显示 `--`
- **位置**：显示当前设置的城市名

台风靠近时有醒目的彩色警告条（绿色=安全、橙色=预警、红色=危险）。

### 顶部工具栏

| 图标 | 功能 |
|------|------|
| ⏰ | 定时和自动调温设置 |
| 🌡️ | 温度规则（决定什么时候开制冷、什么时候开制热） |
| 🔧 | 设备设置（品牌、改名、位置） |
| 📋 | 下载操作日志 |
| 🔍 | 扫描新设备 |
| 🔄 | 刷新数据 |

### 右下角帮助按钮

页面右下角有一个 **❓ 按钮**，点击可以：
- 查看插件功能介绍
- **点击"详细使用文档"直接下载本指南到电脑**
- 查看作者信息和 GitHub 项目地址

## 设备改名

如果你有多台同型号的博联设备，可以通过改名来区分：

1. 在顶部下拉框选择要改名的设备
2. 点击 **🔧 设置**
3. 在"改名"输入框里输入新名字（比如"客厅空调"、"卧室空调"）
4. 点保存

> **注意**：名字不能为空，也不能和其他设备重名。插件会提示你。

## 定时模板（进阶）

### 什么是定时模板？

就是给空调定一个"作息表"。比如：
- **工作日**：早上 8 点自动开，晚上 6 点自动关
- **周末**：早上 10 点自动开，晚上 11 点自动关

每个设备可以绑定不同的模板。你卧室的空调用"周末"，客厅用"工作日"。

### 怎么设置？

1. 点击 ⏰ 打开定时弹窗
2. 打开「启用定时任务」开关
3. 在下拉框里选一个模板，或者点 **+** 新建
4. 勾选生效的日期（比如周一~周五）
5. 填写开机时间和关机时间（可以添加多个时间段）
6. 如果需要自动调温，打开「自动调温」开关
7. 点「保存设置」

### 关闭定时

如果只想用自动调温、不想定时开关：
1. 在弹窗里把「启用定时任务」关掉
2. 保存

定时就停了，但自动调温不受影响。

## 温度规则

### 它是干什么的？

温度规则是"大脑"——告诉插件外面多热该开制冷、多冷该开制热。**定时开关和自动调温都会用到规则。**

### 在哪里设置？

温度规则有独立的编辑入口。点击工具栏 **🌡️ 按钮** 打开规则编辑弹窗。

### 规则怎么写？

一条规则包含四个数：

```
室外 30°C ~ 35°C  → 目标 26°C [制冷]
```

| 部分 | 意思 | 举例 |
|------|------|------|
| 室外温度范围 | 外面在这个温度区间时触发 | 30°C ~ 35°C |
| 目标温度 | 空调开到多少度 | 26°C |
| 模式 | 制冷/制热/除湿/送风/关机 | 制冷 |

### 默认规则

插件自带 6 条默认规则，基本够用：

| 室外温度 | 做什么 |
|---------|--------|
| 36°C 以上 | 制冷 24°C |
| 33°C ~ 35°C | 制冷 25°C |
| 30°C ~ 32°C | 制冷 26°C |
| 25°C ~ 29°C | 制冷 27°C |
| 18°C ~ 24°C | 关机（舒服，不需要空调） |
| 17°C 以下 | 制热 28°C |

> **就像设置恒温器**：你告诉它"外面高于 30 度就给我开 26 度"，剩下的它自己判断。

## 常见问题

### Q: 打开页面显示"插件已停用"？

A: 去 **服务 → Broadlink 空调控制 → 设置**，勾选「启用服务」，保存。取消勾选即可彻底停止插件（所有定时、调温、数据采集都会停止）。

### Q: 扫描不到博联设备？

A: 
1. 检查博联 RM 是否通电（指示灯亮不亮）
2. 确保路由器和博联在同一个网络
3. 多等几秒再试

### Q: 多台同型号设备分不清？

A: 在设备设置弹窗里用"改名"功能给每台起个昵称（见[设备改名](#设备改名)）。插件也会自动给同型号设备加序号区分。

### Q: 能控制空调但温度不对？

A: 检查设置页的「位置」是否正确（经纬度影响天气数据）。点击搜索框输入城市名更新。

### Q: 和风 API 申请后提示 401 错误？

A: 
1. 确认 API Key 没有复制错
2. 确认 Host 填的是和风控制台提供的地址（不是别人的）
3. 新注册的 Key 可能有几分钟延迟

### Q: 台风数据不显示？

A: 台风只在西北太平洋有风暴时才有数据（一般是 6~10 月台风季），其他时间显示 `--` 是正常的。

### Q: 我的空调品牌不在列表里？

A: 目前支持 16 个品牌：格力、美的/华凌/小米（同协议）、海尔、海信、日立、大金、三菱、松下、富士通、奥克斯、巴鲁、开利、现代、Fuego。如果你的品牌不在列表中，程序会自动尝试格力协议。
