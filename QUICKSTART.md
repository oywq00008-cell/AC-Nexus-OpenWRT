# AC-Nexus-OpenWRT 速查

> OpenWRT 路由器端的 Broadlink RM 红外空调控制插件

## 适用设备

| 类型 | 支持情况 |
|------|----------|
| **路由器** | OpenWRT 21.02+，aarch64 / armv7 / x86_64 |
| **红外设备** | Broadlink RM Mini 3 / RM4 Mini / RM Pro+ |
| **空调品牌** | 17 种（格力 / 美的 / 海尔 / 奥克斯 / 大金 / 三菱 / 松下 / 日立 / 富士通 / 海信 / 等）|

## 功能清单

### 1. 远程控制
在 LuCI Web 界面控制空调开关、模式、温度、风速，**不需在路由器前**。

### 2. 天气自动调温
每 2 小时拉一次室外温度（百度/和风双源），按规则自动调目标温度。

### 3. 定时开关
每天指定时间自动开/关空调。

### 4. 台风自动保护
< 100km 时强制关闭所有空调（保护外机被大风吹倒转烧毁 + 防雷击）。

### 5. 设备自动发现
局域网扫描所有 Broadlink RM 设备，下拉切换，多设备并行。

### 6. UCI 配置面板
CBI 设置页填 API key、台风开关、位置（百度地图/和风天气），不用命令行。

### 7. 日志下载
每日 .md 文件记录所有操作，LuCI 工具栏点按钮选日期下载到本地。

### 8. 后台守护
procd 系统服务，开机自启，挂了自动重启，**装了就不用管**。

## 一行总结

> 把 Broadlink RM 插上路由器 → 装上本插件 → 以后空调全自动（按温度、按时间、台风来了自动关）。
